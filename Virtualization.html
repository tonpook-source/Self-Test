Virtualization Solutions Comparison

เปรียบเทียบคุณสมบัติหลักของโปรแกรมจำลองเสมือนสำหรับผู้ใช้งานและองค์กร

คำอธิบายประเภท Hypervisor

Type 1 (Bare-Metal)

ติดตั้งโดยตรงบนฮาร์ดแวร์ของเซิร์ฟเวอร์ (ไม่มี Host OS) ทำงานได้เต็มประสิทธิภาพสูงสุด เหมาะสำหรับระดับองค์กรและ Data Center
(เช่น VMware ESXi, Microsoft Hyper-V on Server)

Type 2 (Hosted)

ติดตั้งเป็นแอปพลิเคชันบนระบบปฏิบัติการหลัก (Host OS) ใช้งานง่าย สะดวก เหมาะสำหรับการใช้งานส่วนตัว, การเรียนรู้ หรือทดสอบระบบ
(เช่น VirtualBox, VMware Workstation)
